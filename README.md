# BlueBlack Sailor Pen Buffet Preview

블루블랙 펜샵 매장 상담용 세일러 펜뷔페 3D 조합 미리보기 웹앱입니다.

- 프로피트 Jr. 기반의 전 세계 공통 파츠 구조 참고
- 캡탑, 캡, 섹션, 배럴, 배럴엔드 5개 수지 파츠 개별 색상 변경
- 닙, 피드, 클립, 캡 밴드, 섹션 링은 고정 부품으로 구현
- 한국 블루블랙 펜샵용 색상 데이터를 구조와 분리하여 관리
- URL 조합 저장 및 복원
- 조합 코드, 링크 복사, 이미지 저장
- 모바일·태블릿·PC 반응형
- GitHub Pages 자동 배포 구성

공식 로고, 상표, 각인은 포함하지 않습니다. 공개된 공식 파츠 안내와 제품 사진을 참고한 매장 상담용 중립 모델입니다.
