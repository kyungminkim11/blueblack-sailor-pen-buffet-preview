공식 페이지에서 확인된 내용

공식 페이지에는 총 12가지 글꼴과 각 글꼴이 지원하는 문자 종류가 안내되어 있습니다.

12각인 글꼴

돋움부터 모노타입까지 12가지 선택지가 있습니다.

4지원 문자 종류

한글, 한문, 영어, 숫자 지원 여부가 글꼴마다 다릅니다.

!필기체 주의

필기체는 영어와 숫자를 지원하지만 연속 대문자는 사용할 수 없습니다.

각인 전 확인 순서

아래 순서는 매장 상담을 빠르게 진행하기 위한 확인 흐름입니다.

01제품 확인

선택한 제품의 각인 가능 여부와 위치를 직원에게 확인합니다.

02글꼴 선택

입력할 문자 종류를 지원하는 글꼴을 선택합니다.

03문구 확인

철자, 띄어쓰기, 대문자와 소문자를 그대로 확인합니다.

04최종 확인

직원과 각인 내용 및 제품별 적용 가능 여부를 최종 확인합니다.

꼭 확인해 주세요

  • 공식 페이지에는 가격, 글자 수, 소요 기간이 별도로 안내되어 있지 않습니다.
  • 제품마다 각인 가능 여부와 위치가 다를 수 있으므로 직원 확인이 우선입니다.
  • 완료 후 수정이 어려울 수 있으므로 철자와 대소문자를 신중하게 확인해 주세요.
  • 필기체는 연속 대문자를 사용할 수 없습니다.
