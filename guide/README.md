# BlueBlack Fountain Pen Guide v4

블루블랙 펜샵 고객이 매장에서 직접 사용하는 만년필 검색·추천·문제 해결 웹앱입니다.

## 고객 기능

- 10문항 점수식 첫 만년필 추천
- 28개 비교 후보
- 10·20·30·40·50·60만원대별 추천
- 16개 사용 상황별 추천
- 공식몰 확인 가격과 기준일 표시
- 오타·유사어·초성 대응 검색
- 앱 내부 상세 가이드
- 안전 중심 문제 해결 분기
- 상담 메모와 임시 접수번호 생성
- 글자 확대, 고대비, 키보드 접근성
- 키오스크 자동 초기화
- 오프라인 기본 가이드와 업데이트 알림

## 상황별 추천

1. 완전한 입문용
2. 첫 상위 모델
3. 학생·시험 필기
4. 직장인 데일리
5. 장시간 필기
6. 왼손잡이
7. 다이어리·잉크 표현
8. 병 잉크 중심
9. 휴대·출장용
10. 10만원대 선물
11. 격식 있는 비즈니스 선물
12. 클래식 디자인
13. 종이 감각이 느껴지는 필기
14. 한정판·컬렉션
15. 기존 사용자 업그레이드
16. 서명·큰 글씨

## 가격대 정의

- 10만원대: 100,000–199,999원
- 20만원대: 200,000–299,999원
- 30만원대: 300,000–399,999원
- 40만원대: 400,000–499,999원
- 50만원대: 500,000–599,999원
- 60만원대: 600,000–699,999원

## 실행 주소

- 일반 고객: `/guide/`
- 키오스크: `/guide/?mode=kiosk`
- 입구 QR: `/guide/?source=entrance`
- 시필대 QR: `/guide/?source=nib-table`
- 잉크 코너 QR: `/guide/?source=ink-zone`
- 카운터 QR: `/guide/?source=counter`

## 데이터 구성

### `catalog.js`

- 3만–5만원대 입문 상품
- 기본 사용·관리·문제 해결 가이드
- 기본 10문항 추천 데이터

### `premium.js`

- 10만–60만원대 공식몰 확인 후보
- 가격대 정의와 설명
- 16개 상황별 추천 후보와 상담 포인트

### `extensions.js`

- 상황별 추천 섹션 생성
- 가격대별 탭과 제품 카드 생성
- 가격대·상황 검색어 연결
- 제품 상세 비교와 직원 상담 메모 생성

## 데이터 원칙

- 표시 가격은 `verifiedAt` 기준일과 함께 제공합니다.
- 할인, 옵션, 온라인 판매 상태와 매장 재고는 구분합니다.
- 공식 페이지 품절 상품은 품절·재입고 확인 필요 상태로 표시합니다.
- 매장 재고를 추측하지 않습니다.
- 추천 순서는 절대적인 성능 순위가 아니라 해당 상황의 비교 시작점입니다.
- 최종 제품 선택은 실제 시필을 기준으로 합니다.
- 고객 연락처와 상담 본문은 로컬 이용 통계에 저장하지 않습니다.

## 추천 로직

기본 추천은 사용 경험, 목적, 글씨 크기, 종이, 필압, 사용 손, 무게, 충전 방식, 예산과 디자인 취향을 비교합니다.

상황 추천은 고객이 매장에서 자주 말하는 표현을 기준으로 구성합니다.

- “처음 써요”
- “10만원대 선물이에요”
- “하루 종일 필기해요”
- “왼손잡이예요”
- “병 잉크를 많이 써요”
- “업무용으로 고급스러운 걸 원해요”
- “현재 펜보다 업그레이드하고 싶어요”

## 안전 원칙

- 닙 변형, 균열, 비정상 누출은 즉시 사용 중단으로 분기합니다.
- 닙 벌리기, 금속 도구, 뜨거운 물, 세제, 알코올 사용을 안내하지 않습니다.
- 수리 비용, 보증 적용, 실시간 재고를 단정하지 않습니다.

## 배포 전 검사

1. `node --check guide/app.js`
2. `node --check guide/extensions.js`
3. `node --check guide/data/catalog.js`
4. `node --check guide/data/premium.js`
5. `node --check guide/modules/search.js`
6. `node --check guide/modules/recommendation.js`
7. 10–60만원대 탭별 제품 표시 확인
8. 16개 상황의 후보 3종 연결 확인
9. 모바일과 태블릿에서 추천 완료 확인
10. 오프라인 전환 후 가격대·상황 데이터 확인
11. 공식몰 링크와 가격 기준일 확인
12. 공개 Notion 페이지에 내부 DB가 포함되지 않았는지 확인

## 현재 의도적으로 제외한 기능

- 실시간 POS·재고 자동 연동
- 직원 휴대전화 자동 푸시 알림
- 검수되지 않은 영문·일문 화면
- 사용 권한이 확인되지 않은 공식 로고·상품 사진

위 기능은 관련 계정, API와 승인된 자산이 제공된 이후 연결합니다.
