상품 데이터 업데이트

이카운트에서 내려받은 상품·바코드 파일과 재고현황 파일을 선택하면 상품 빠른찾기와 재고 조사 데이터가 함께 갱신됩니다.

1. 파일 두 개 선택품목등록 파일과 재고현황 파일을 각각 선택합니다.
2. 행 수 확인파일 형식과 품목코드 일치 수를 자동으로 확인합니다.
3. DB 업데이트버튼을 누르면 Supabase 상품 DB에 실제로 반영됩니다.
