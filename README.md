# BlueBlack Pen Shop Digital Guide

블루블랙 펜샵 매장에서 고객과 직원이 함께 사용하는 모바일 우선 디지털 상담 가이드입니다.

## 주요 화면

- 세일러 펜뷔페 3D 조합 미리보기
- 잉크 소분 및 본병 검색
- 매장 안내도와 브랜드 위치 검색
- 각인 글꼴·문구 확인
- 브랜드별 A/S 접수처 안내
- 영수증 포토리뷰 이벤트
- 블루블랙 최신 소식과 공식 채널
- 고객 안내 / 직원 빠른 도구 모드

## 펜뷔페

색상 선택 대상은 6개 파츠입니다.

1. `cap_top`
2. `cap_body`
3. `grip_section`
4. `center_ring_or_connector`
5. `barrel_body`
6. `barrel_end`

기존 URL과의 호환을 위해 `cap_end`, `nib_grip`, `metal_parts` 키도 함께 지원합니다.

- 파츠별 독립 색상 변경
- 3D 회전·확대·축소
- 파츠 분리 보기
- 조합 코드 생성
- URL 저장 및 복원
- 링크·이미지 공유
- 직원 확인 화면
- WebGL 실패 시 정적 미리보기 제공
- `/models/sailor-pen-buffet.glb`가 있으면 GLB 모델을 우선 적용하고, 없거나 노드명이 맞지 않으면 절차형 모델을 사용

GLB 모델은 다음 노드 또는 mesh 이름을 사용합니다.

- `cap_top`
- `cap_body`
- `grip_section`
- `center_ring_or_connector`
- `barrel_body`
- `barrel_end`

## 공통 UX

- 모바일·태블릿·PC 반응형
- 한국어, 영어, 일본어, 중국어 간체·번체, 베트남어, 인도네시아어, 태국어 선택
- 공통 언어 메뉴와 큰 터치 영역
- 고객용 핵심 기능 우선 배치
- 직원용 새 고객 시작 및 빠른 상담 링크
- 최근 사용 도구 저장
- 포토리뷰 모바일 고정 버튼
- 뉴스 데이터 로컬 캐시와 마지막 업데이트 표시
- 잉크 검색 UI 네트워크 실패 시 캐시 복구
- 접근성 포커스와 reduced-motion 대응

## 운영 원칙

이 프로젝트는 쇼핑몰이 아닙니다. 회원가입, 로그인, 결제, 장바구니, 주문 기능을 제공하지 않습니다.

공식 로고, 상표, 각인은 임의로 포함하지 않으며 매장 상담과 조합 미리보기에 집중합니다.
